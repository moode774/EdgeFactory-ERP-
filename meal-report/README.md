# 🌲 Forest Edge - نظام تقارير الوجبات الرسمي

نظام احترافي متكامل لإدارة وطباعة تقارير استلام الوجبات اليومية مع تصميم راقي وشعار Forest Edge.

## ✨ المميزات الرئيسية

### 🎨 تصميم احترافي فاخر
- شعار Forest Edge مصمم بعناية مع رمز الشجرة
- تدرجات لونية خضراء أنيقة (#2d5016, #4a7c2c)
- تصميم نظيف ومنظم مع علامة مائية خفيفة
- خطوط عربية راقية (Cairo & Amiri)

### 📋 نظام تقارير شامل
- معلومات التوصيل الكاملة (التاريخ، الوقت، المركبة)
- بيانات المورد والسائق
- جدول تفصيلي لفئات الوجبات مع الألوان
- حساب تلقائي للإجمالي
- قسم للملاحظات الإضافية
- منطقة التوقيعات والاعتماد

### 🎛️ لوحة تحكم متقدمة
- تعديل جميع البيانات بسهولة
- إضافة وحذف فئات الوجبات
- اختيار ألوان مخصصة لكل فئة
- معاينة مباشرة للتغييرات
- حفظ وطباعة فورية

### 🖨️ جاهز للطباعة
- تنسيق A4 Portrait مثالي
- تحسين تلقائي للطباعة
- إخفاء عناصر التحكم عند الطباعة
- جودة عالية للمستندات الرسمية

## 🚀 كيفية الاستخدام

### 1. فتح الملف
افتح ملف `index.html` في المتصفح مباشرة

### 2. تعديل البيانات
- اضغط على زر "تعديل البيانات"
- قم بتحديث معلومات التوصيل
- عدّل أعداد الوجبات لكل فئة
- أضف أو احذف فئات حسب الحاجة
- أضف ملاحظات إضافية

### 3. الطباعة
- اضغط على زر "طباعة التقرير الرسمي"
- تأكد من إعدادات الطباعة (A4 Portrait)
- اطبع أو احفظ كـ PDF

## 🎯 الفئات الافتراضية

1. **جنوب آسيا** (برتقالي) - باكستان، الهند، بنغلاديش
2. **الفلبين** (أزرق) - العمالة الفلبينية
3. **مصر** (أحمر) - العمالة المصرية
4. **السعودية** (أخضر) - الموظفون السعوديون

4. احفظ نسخة PDF للأرشفة

## 🌟 مميزات التصميم

- **الشعار**: تصميم شجرة احترافي مع دوائر زخرفية
- **الألوان**: تدرجات خضراء طبيعية تعكس هوية Forest Edge
- **الجداول**: تصميم نظيف مع تمييز الصفوف
- **التوقيعات**: مناطق واضحة للتوقيع والاعتماد
- **الختم**: منطقة مخصصة للختم الرسمي
